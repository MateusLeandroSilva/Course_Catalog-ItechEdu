# Catálogo de Cursos - Sistema de Consulta

## Sobre o Projeto

Este projeto está sendo desenvolvido durante meu estágio como parte de uma atividade prática proposta pela empresa. O objetivo é criar um sistema simples e funcional para facilitar a consulta de informações sobre os pacotes de cursos oferecidos.

## Descrição

O sistema apresenta um catálogo interativo de 7 pacotes de cursos profissionalizantes, permitindo que os usuários consultem rapidamente informações detalhadas sobre cada pacote, incluindo:

- Cursos incluídos no pacote
- Carga horária total
- Duração em meses
- Valor total do investimento

## Como Executar

1. Compile o arquivo `.java`:
```bash
javac javaapplication2/JavaApplication2.java
```

2. Execute o programa:
```bash
java javaapplication2.JavaApplication2
```

3. Digite o código do pacote desejado (1-7) para visualizar as informações
